University of British Columbia Software Development Competition

With a team of 3 other engineers I competed in the 2021 UBCO software development competition. The competition lasted from 8 a.m. to 6 p.m. At the beggining of the competition we where givin a problem and tasked with not only solving the problem but developing a function UI for the solution. You can see on the left the code we wrote in python using tkinter to make our UI.

The problem was very similar to that of a real world minesweeper situation where there is areas of radiation and a scanner that can find out how close that radiation is. My team worked over the cource of 5 hours to attempt to create a working simulation of the situation and and a user interface that could be used for training purposes. To the right you can the see the orignal python code traslated to Java for the UI using JSwing.

Givin more time we would have been able to create a fully functional system. But in the five hours of worktime provided we where able to create a basic UI with function code to simulate the situation. With out work we got 2nd place and we will be going on to the Larger Western Engineering Compitition this January to compete again.
